# Estudo Fila

## Descrição

Este repositório possui um pequeno programa na linguagem C que visa praticar a implementação da estrutura de dados fila.

O programa é algo extremamente simples, ele cria uma fila de números inteiros, adiciona alguns e faz algumas ações de remoção.

## Estrutura dos arquivos

A estrutura dos arquivos é bem simples.

- src: possui todos os arquivos .c do programa;
- include: possui todos os arquivos de header (.h) do programa;

A estutura do código-fonte é bem simples, na raiz há um Makefile para compilação, além de algumas configurações de debugging na pasta .vscode

## Execução

Para apenas compilar, basta executar o seguinte comando:

```bash
make
```

Um binário chamado "main.exe" será gerado na raiz do projeto. Para executar, basta usar o seguinte comando:

```bash
make run
```

Para limpar todos os arquivos de build, execute o seguinte comando:

```bash
make clean
```
